# homebrew-clamui

Homebrew tap for [clamui](https://murilo-cunha.github.io/clamui-site/) — a markdown
editor with Claude built in.

```bash
brew install murilo-cunha/clamui/clamui
```

Apple silicon only. The cask is bumped automatically on every release; versions and
checksums point at the public [clamui-site releases](https://github.com/murilo-cunha/clamui-site/releases).
